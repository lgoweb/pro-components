    安装

    当前 ProComponents 每一个组件都是一个独立的包，你需要在你的项目中安装对应的 npm 包并使用。

    $ npm i @lekp/pro-foo --save

    当前 ProComponents 提供了如下组件可直接使用：

    • npm i @lekp/pro-foo --save

    我们所有的包都使用 less 来进行样式管理，方便进行主题的自定义。如果你没有 less-loader 可以尝试从 dist 中导入 css。
